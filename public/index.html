Timestamp Microservice

The timestamp microservice project is a full stack JavaScript app that provides an API endpoint (/api/:date) for converting Unix timestamps (a number representing the number of milliseconds since January 1, 1970) and common date formats (e.g. mm-dd-yyyy). A JSON object is returned with the corresponding Unix timestamp and UTC date string of the date parameter requested.


For example, both requests to
https://timestamp-microservice-sf.glitch.me/api/1451001600000
https://timestamp-microservice-sf.glitch.me/api/12-25-2015
will return


{ "unix": 1451001600000, "utc": "Fri, 25 Dec 2015 00:00:00 GMT" }

    Examples of other valid url encoded date formats:
  • 12/25/2015: 12%2F25%2F2015
  • Dec 25 2015: %22Dec%2025%202015
  • 25 Dec 2015: 25%20Dec%202015
  • 2015-12-25: 2015-12-25
  • Fri, 25 Dec 2015: Fri%2C%2025%20Dec%202015
  • December 25, 2015: December%2025%2C%202015

    • *There are plenty more valid formats as long as it is properly encoded for urls*

This project is designed based on requirements for freeCodeCamp's Backend Development and APIs Certification.
